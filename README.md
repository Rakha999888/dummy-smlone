# SMLONE Student Portal

Portal Siswa **SMLONE** adalah aplikasi web modern dan minimalis yang dirancang untuk memberikan akses yang mudah dan terpusat ke materi pelatihan bagi para trainee. 

## 🚀 Fitur Utama

- **Autentikasi Aman:** Sistem login menggunakan ID Siswa dan Password.
- **Dashboard Personal:** Menampilkan informasi siswa dan tautan akses pelatihan yang spesifik untuk setiap pengguna.
- **Desain Responsif:** Tampilan yang rapi dan optimal diakses melalui perangkat desktop maupun handphone (Split-screen UI).
- **Modern UI/UX:** Antarmuka pengguna yang bersih menggunakan palet warna biru yang profesional.

## 🛠️ Teknologi yang Digunakan

- **Frontend:** React, TypeScript
- **Styling:** Tailwind CSS
- **Build Tool:** Vite
- **Ikon:** Lucide React
- **Routing:** React Router DOM

## ⚙️ Cara Menjalankan Proyek Secara Lokal

Ikuti langkah-langkah di bawah ini untuk menjalankan proyek di komputer Anda:

1. **Clone repositori**
   ```bash
   git clone https://github.com/Rakha999888/dummy-smlone.git
   cd dummy-smlone
   ```

2. **Instal dependensi**
   ```bash
   npm install
   ```

3. **Jalankan server pengembangan**
   ```bash
   npm run dev
   ```

4. Buka browser dan akses URL lokal yang disediakan oleh Vite (biasanya `http://localhost:5173`).
